📘 Guide pratique – BENYS

Comment utiliser l'application étape par étape

🔐 Se connecter à l’application

Arrivez sur la page d’accueil “Welcome to BENYS”

  • Entrez votre nom d’utilisateur
  • Entrez votre mot de passe
  • Cliquez sur Login

👉 Vous êtes redirigé vers votre Dashboard

📊 Comprendre le Dashboard

  • 📁 À gauche → vos projets
  • 📝 À droite → vos tickets assignés

👉 Cliquez sur :

  • un projet → voir ses détails
  • un ticket → voir/modifier ses informations

📂 Accéder à ses projets

Cliquez sur Projects (en haut)

👉 Vous voyez tous vos projets

👉 Cliquez sur un projet pour l’ouvrir

⚠️ Permissions

Utilisateur standard :

  • ✔ Voir projets
  • ✔ Voir tickets

👉 L’admin doit vous ajouter à un projet pour plus de droits

👑 Guide Admin

➕ Créer un projet

  • Aller dans Projects
  • Cliquer sur + New Project
  • Remplir les informations
  • Valider

👥 Gérer les membres

  • Ouvrir un projet
  • Aller dans Manage Members
  • Ajouter un utilisateur
  • Choisir un rôle :
    • Contributor
    • Read-only

❌ Supprimer un utilisateur

👉 Résultat :

  • L’utilisateur perd l’accès
  • Ses tickets restent mais deviennent non assignés

🏃‍♂️ Créer et gérer un Sprint

➕ Créer

  • Ouvrir un projet
  • Cliquer sur Create Sprint
  • Nom + objectif + date début

👉 Fin automatique (14 jours), possibilité de changer la durée du sprint

▶️ Start

👉 Le sprint devient actif

⛔ Close

👉 Sprint terminé définitivement

🎫 Créer un Ticket

  • Type : Bug / Task / Story
  • Titre + description
  • Options :
    • Sprint
    • Epic
    • Ticket lié
    • Priorité
    • Workload
    • Assigné

📌 Assigner

Choisir un utilisateur dans Assignee

📦 Product Backlog

  • Voir tâches non planifiées
  • Organiser les priorités

📊 Kanban (Sprint)

Colonnes :

  • 🆕 New
  • 🔄 Active
  • ✅ Closed

👉 Glisser-déposer les tickets

🎯 Epic

  • Créer un epic
  • Ajouter couleur + dates
  • Regrouper plusieurs tickets

📑 Navigation rapide

  • Overview → résumé
  • Backlog → tâches
  • Active Sprint → kanban
  • Issues → tickets
  • Epic → epics

👤 Profil

  • Nom
  • Email
  • Photo
